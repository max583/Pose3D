## 2. Drag-and-Drop суставов (ИСПРАВЛЕНО)

### 2.1 Базовое перетаскивание

- [ ] Наведение на точку скелета меняет курсор на **"grab"** 👆
- [ ] Нажатие на точку начинает перетаскивание:
  - [ ] Курсор меняется на **"grabbing"** ✊
  - [ ] Точка "подсвечивается" (белый emissive)
  - [ ] Внизу появляется надпись **"✋ Dragging joint..."** (оранжевый, пульсирует)
- [ ] Движение мыши перемещает точку:
  - [ ] Точка двигается за курсором
  - [ ] Кости перерисовываются автоматически
  - [ ] Движение плавное (без рывков)
- [ ] Отпускание кнопки завершает перетаскивание:
  - [ ] Курсор возвращается к "grab" или "default"
  - [ ] Надпись "Dragging joint..." исчезает
  - [ ] OrbitControls включается обратно

### 2.2 Чувствительность drag

- [ ] Движение мыши на 100px ≈ смещение на 1 единицу 3D
- [ ] Чувствительность комфортная (не слишком быстро/медленно)
- [ ] Точки не "прыгают" при начале перетаскивания
- [ ] Точки возвращаются на место при наведении (нет смещения)

### 2.3 OrbitControls взаимодействие

- [ ] **Во время drag** OrbitControls отключён:
  - [ ] Вращение камеры не работает
  - [ ] Зум не работает
  - [ ] Пан не работает
- [ ] **После drag** OrbitControls включён:
  - [ ] Вращение работает (левая кнопка мыши)
  - [ ] Зум работает (колёсико мыши)
  - [ ] Пан работает (правая кнопка мыши)

### 2.4 Все 25 точек

- [ ] Нос (Nose) - жёлтая

## 5. Экспорт для ControlNet

### 5.1 Экспорт JSON

- [ ] Кнопка **"Export JSON"** работает:
  - [ ] Скачивается файл `pose_openpose.json`
  - [ ] Файл открывается в текстовом редакторе
  - [ ] Содержимое соответствует формату OpenPose:
    ```json
    {
      "version": 1.3,
      "people": [{
        "pose_keypoints_2d": [x0, y0, c0, ...]
      }]
    }
    ```
  - [ ] `pose_keypoints_2d` содержит 75 значений (25 точек × 3)
  - [ ] Каждые 3 значения: x, y, confidence
  - [ ] Confidence = 1 для всех точек

### 5.2 Экспорт PNG 512x512

- [ ] Кнопка **"Export PNG (512)"** работает:
  - [ ] Скачивается файл `pose_openpose.png`
  - [ ] Файл открывается в просмотрщике изображений
  - [ ] Размер изображения: 512×512 пикселей
  - [ ] Фон чёрный (#000000)
  - [ ] Кости цветные (соответствуют 3D цветам)
  - [ ] Точки белые (суставы)
  - [ ] Скелет центрирован

### 5.3 Экспорт PNG 1024x1024

- [ ] Кнопка **"Export PNG (1024)"** работает:
  - [ ] Скачивается файл `pose_openpose_1024.png`
  - [ ] Размер изображения: 1024×1024 пикселей
  - [ ] Фон чёрный
  - [ ] Кости и точки видны (выше разрешение)

### 5.4 Экспорт после редактирования

- [ ] Перетащите несколько точек
- [ ] Экспортируйте JSON:
  - [ ] Координаты изменились (не равны пресету)
  - [ ] Данные соответствуют текущей позе
- [ ] Экспортируйте PNG:
  - [ ] Изображение отражает текущую позу
  - [ ] Точки сместились от начальной позиции

### 5.5 Экспорт разных пресетов

- [ ] Выберите "T-Pose" → Экспорт JSON → проверьте координаты рук
- [ ] Выберите "Sitting" → Экспорт JSON → проверьте координаты ног (должны быть ниже)
- [ ] Координаты различаются между пресетами

  - [ ] `backend.log` - основные события

  - [ ] `backend_pose.log` - операции с позами
- [ ] Записи в логах содержат:
