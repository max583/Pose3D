# Plans

Актуальный рабочий план проекта находится в [`../PLAN.md`](../PLAN.md).

Эта папка больше не является основным backlog. Она содержит только навигацию и архив исторических планов.

## Актуальные документы

- [`../PLAN.md`](../PLAN.md) — текущий roadmap и рабочий протокол.
- [`../STATUS.md`](../STATUS.md) — текущий статус продукта.
- [`../ai/docs/feature-task-template.md`](../ai/docs/feature-task-template.md) — шаблон задачи для фичи.
- [`../ai/docs/r3f-smoke-manual-checklist.md`](../ai/docs/r3f-smoke-manual-checklist.md) — smoke/manual чек-листы для R3F.
- [`../ai/decisions/README.md`](../ai/decisions/README.md) — ADR и архитектурные решения.

## Архив

Исторические планы перенесены в [`archive/`](archive/). Они сохраняются только как контекст прошлых решений и не должны использоваться как источник актуального roadmap.

Если архивный документ противоречит `PLAN.md`, `STATUS.md` или коду, считать актуальными `PLAN.md`, `STATUS.md` и код.
